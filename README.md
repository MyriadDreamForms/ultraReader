# ultraReader

Offline Manga/Webtoon Reader built with .NET MAUI. Supports Android, iOS, macOS, and Windows.

## 🔥 Features
- Read Manga and Webtoons offline
- Supports multiple formats (e.g., CBZ, Images)
- Bookmarking system
- Cross-platform UI with responsive layout
- Dark/Light Mode

## 📦 Platforms
- ✅ Windows
- ✅ Android
- ✅ macOS
- ✅ iOS
